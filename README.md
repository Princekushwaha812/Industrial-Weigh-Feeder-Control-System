# Industrial-Weigh-Feeder-Control-System
Monitoring and control logic for industrial weigh feeders using load cells, tachometers, and VFD frequency modulation.
Industrial Weigh Feeder Monitoring & Control SystemOverviewThis project details the design and operational logic of an industrial Weigh Feeder system, a critical component in cement and mining industries for the continuous, weight-based feeding of raw materials. The system utilizes a closed-loop control architecture to ensure consistent material flow.System ArchitectureThe system operates by integrating mechanical load sensing with rotational speed measurement to calculate real-time mass flow.Weight Measurement: Uses strain-gauge based load cells mounted beneath the conveyor frame to convert material force into an electrical signal.Speed Measurement: A tachometer/rotary encoder measures pulley RPM via magnetic or optical pulses.Control Logic: A PLC or DCS processes these inputs to calculate Tons Per Hour (TPH) and adjusts the motor speed via a Variable Frequency Drive (VFD).Mathematical FoundationTo maintain precision, the system uses the following calculations:1. Flow Rate (TPH):$$\text{TPH} = \text{Belt Load (kg/m)} \times \text{Belt Speed (m/s)} \times 3.6$$2. Signal Mapping (4-20 mA):
The calculated TPH is mapped to an industry-standard analog signal for DCS communication.
$$\text{Output Current (mA)} = 4 + \left( \frac{\text{Current TPH}}{\text{Max TPH}} \right) \times 16$$3. Frequency Modulation:
The VFD adjusts frequency proportionally to reach target flow rates.
$$\text{New Frequency} = \text{Current Frequency} \times \left( \frac{\text{Target TPH}}{\text{Current TPH}} \right)$$
